# 大数据学习路线

## 第一阶段：基础入门
- Linux 基础、Shell 脚本
- SQL 进阶（窗口函数、复杂查询、执行计划）
- Java/Scala/Python 基础

## 第二阶段：离线计算
- Hadoop、HDFS、YARN
- Hive 与数据仓库建模（维度建模、分层）
- Spark Core、Spark SQL

## 第三阶段：实时计算
- Kafka 消息队列
- Flink 流处理（DataStream、Table API、SQL）

## 第四阶段：进阶体系
- 调度系统（DolphinScheduler、Airflow）
- 数据治理（元数据、血缘、质量、权限）
- 湖仓技术（Iceberg、Hudi、Delta Lake）

## 学习建议
- 每个阶段做一个小项目闭环
- 从业务需求出发，倒推技术学习
- 笔记和复盘是学习效率的关键
